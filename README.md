# 🧟 Zombie Purge: Xonqa Edition

C++ va SFML kutubxonasi yordamida yaratilgan 2D shooter o'yini. Bu loyiha dasturlash mantiqi, grafik elementlar bilan ishlash va real-vaqtda to'qnashuvlarni hisoblash asosida qurilgan.

## 🌟 Xususiyatlari

- **Double Control:** O'yinni ham `W/S`, ham yo'nalish ko'rsatkichlari (`Arrow Keys`) orqali boshqarish mumkin.
- **Dynamic Difficulty:** Ballar ortishi bilan zombilar tezlashadi va chiqish vaqti qisqaradi (spawn rate).
- **Parallax Background:** Cheksiz va harakatlanuvchi fon effekti.
- **Zombie Wobble:** Zombilarning sinus funksiyasi yordamida titrab kelishi (haqiqiy zombi effekti).

## 🎮 Boshqaruv

- **Tepaga:** `W` yoki `↑` (Up Arrow)
- **Pastga:** `S` yoki `↓` (Down Arrow)
- **O'q otish:** `Space` (Bo'shliq tugmasi)
- **Qayta boshlash:** `R` (Game Over-dan keyin)

## 🛠️ Texnologiyalar

- **Til:** C++
- **Kutubxona:** SFML (Simple and Fast Multimedia Library)
- **Tizim:** Windows 11 24H2
- **Compiler:** GCC (MinGW-w64)
